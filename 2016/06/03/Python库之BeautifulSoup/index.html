Python库之BeautifulSoup

发表于 2016-06-03   |  

简介

BeautifulSoup使我们常用于解析html的一个python的第三方库,可以通过pip安装

pip install beautifulsoup4

使用方法

创建BeautifulSoup

soup = BeautifulSoup(html)

也可以使用本地文件

soup = BeautifulSoup(open(‘index.html’))

格式化输出 soup.prettify()

四大对象种类

  • Tag
  • NavigableString
  • BeautifulSoup
  • Comment

Tag

简单来说,即为HTML中的标签

soup.title

soup.head

soup.a

这种方法可以迅速获得标签的内容,不过它查找的是第一个复合要求的标签

对于Tag有两个重要的属性 即name 和 attrs

name即为标签本身

attrs为所有值得一个字典

NavigableString

获得标签的内容后,通过.string来获取标签内部的文字

BeautifulSoup

BeautifulSoup 对象表示的是一个文档的全部内容.大部分时候,可以把它当作Tag 对象，是一个特殊的 Tag，我们可以分别获取它的类型，名称，以及属性来感受一下

Comment

Comment 对象是一个特殊类型的 NavigableString 对象，其实输出的内容仍然不包括注释符号，但是如果不好好处理它，可能会对我们的文本处理造成意想不到的麻烦。

执行字符串前,先判断是不是comment对象,是的话代表被注释

CSS选择器

类似jQuery的选择器

使用select

组合查找 soup.select(‘p #link1’)

子标签查找 soup.select(“head > title”)

属性查找 soup.select(‘a[class=”sister”]’)
