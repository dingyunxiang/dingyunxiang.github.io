git命令大全

发表于 2016-07-15   |  

创建版本库

版本控制与查看

  • git status

查看仓库当期的状态

  • git diff

查看与上次修改的区别

  • git add

将修改提交到仓库

  • git commit -m “*“

将仓库提交到远程仓库

版本回退

  • git log (—pretty=oneline)

显示日志

  • git reset –hard HEAD(当前版本,每个^往回以多功能一个版本,HEAD~5)
  • git reflog 查看多有操作的id(重返未来)

工作区和暂存区的概念

git add将所有修改的文件放到暂存区,然后执行commit之后就可以一次性的将所有暂存区的修改提交分支

管理修改

  • git checkout - - file可以丢弃工作区的修改
  • git reset HEAD file 可以将暂存区的修改回退到工作区,然后再执行checkout

删除文件

  • git rm ——> git commit

关联远程库

git remote add origin git@….
